🌐 Publish Your Website with GitHub Pages

A step-by-step guide to go from an HTML file to a live, shareable public website — for free.

1

Create a GitHub Account

Go to github.com and click Sign up. Enter your email, create a password, and choose a username. This username will appear in your final website URL.

GitHub's sign-up page — enter your details and verify your email to activate your account.

2

Create a New Repository

A repository is like a folder that stores your website files on GitHub. To create one:

  • Click the + icon in the top-right corner
  • Select "New repository"
  • Give it a name (e.g. my-website)
  • Set visibility to Public
  • Click "Create repository"

Click Repositories tab on your profile, then click the green New button (top right) to create a new repository.

💡 The repository name will also appear in your URL: username.github.io/repo-name/
3

Upload Your HTML File

Inside your new repository, upload your HTML file:

  • Click "Add file" → "Upload files"
  • Drag and drop your .html file or click to browse
  • Scroll down and click "Commit changes"

4

Rename the File to index.html

GitHub Pages serves index.html as the default page. Rename your file so it loads automatically when someone visits your URL:

  • Click on your uploaded file in the repository
  • Click the pencil ✏️ icon to edit
  • Change the filename at the top to index.html
  • Click "Commit changes"
github.com/gonzalobalmori/my-website/edit/main/my-website.html
Rename file: my-website.html → index.html ✓
⚠️ Renaming to index.html makes this your homepage
Commit changes

Edit the filename field at the top of the editor — change it to index.html and commit.

💡 If you skip this step, your file is still accessible at the full URL ending in /yourfilename.html
5

Enable GitHub Pages

Now tell GitHub to serve your repository as a website:

  • Go to your repository and click the Settings tab
  • In the left sidebar, click Pages
  • Under "Source", select Deploy from a branch
  • Set branch to main and folder to / (root)
  • Click Save

6

Your Website is Live! 🎉

After about 1–2 minutes, go back to Settings → Pages and you'll see a green banner with your live URL:

github.com/gonzalobalmori/my-website/settings/pages
✅ Your site is live at
https://gonzalobalmori.github.io/my-website/

GitHub confirms your site is live with a green banner and a clickable link.

🔗 https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/

Share this link with anyone — no GitHub account needed to view it. It works in any browser, on any device, just like a regular website.

✏️ To update your site: just edit index.html on GitHub and commit. Changes go live automatically within a minute or two.
